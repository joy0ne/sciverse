# Sciverse

> Interactive Science Explorer — 33,603 topics, 11 categories, 1,327 curated Newton-level articles, fully offline single-file HTML5 web app.

## What is this

Sciverse is a free, offline-first science education web app. Open `Sciverse.html` in any modern browser and explore physics, chemistry, math, electrical engineering, biology, geography, computing, quantum, neuroscience, and cross-domain science — with interactive canvas simulations on every topic and an offline study assistant that answers from the built-in library (no network calls, no data leaves your device).

**No tracking. No ads. No external requests. No account.**

## Files

- `Sciverse.html` — the entire app, ~35 MB single file. This is the only file you actually need.
- `index.html` — landing page that redirects to `Sciverse.html` (so GitHub Pages root works).
- `manifest.json` — PWA manifest (lets users install as an app from the browser).
- `sw.js` — service worker (offline caching).
- `icons/` — PWA icons (192/512/Apple touch/favicon).

## How to use

### Just read it locally
Open `Sciverse.html` in your browser. Done.

### Install as an app on phone/desktop (no app store needed)
1. Open the GitHub Pages URL on your phone.
2. Android (Chrome): menu → **Install app** / **Add to Home screen**.
3. iOS (Safari): share button → **Add to Home Screen**.
4. The app installs offline-capable with its own icon — works like a native app.
